# Security Policy

## Supported versions

Security fixes are provided for the latest released version.

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |
| Older   | Best effort |

## Reporting a vulnerability

Please report security issues privately using GitHub's private vulnerability reporting feature on this repository.

Include:

- Affected version or commit
- Reproduction steps
- Impact assessment
- Any suggested mitigation

Please do not open a public issue for suspected vulnerabilities until the issue has been reviewed.

## Scope

This library profiles LLM API calls and analyzes token usage. It does not store API keys, manage cloud credentials, or proxy requests to LLM providers.

If a vulnerability depends on a specific runtime environment, cloud platform, or LLM provider SDK, include those environment details in the report.
